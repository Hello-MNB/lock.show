## 4. שורה תחתונה — כמה עולה להריץ פיילוט

**"פיילוט רזה"** (נשארים על Free ב-Vercel/Supabase/Resend/Tavily):
≈ **$5–20/חודש** (רק Anthropic) + **~$30/שנה** דומיין.

**"פרודקשן מוקשח"** (Vercel Pro + Supabase Pro):
≈ **$45–65/חודש** + Anthropic $5–20 + **~$30/שנה** דומיין. Resend/Tavily עדיין בחינם בנפח פיילוט.

*אף אחד מהחיבורים שביצענו היום (Anthropic-חשבון, YouTube, Spotify, GA4, SoundCloud, Google-OAuth, Tavily) לא עלה שקל. העלויות האמיתיות מתחילות רק בדומיין + הקשחת-פרודקשן.*
