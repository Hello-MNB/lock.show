Non-negotiable responsive rules

01 · Touch

44px minimum.

Every mobile control, navigation item and dismiss target meets the minimum physical target.

02 · Reading

One story per viewport.

Progressive disclosure replaces compressed dashboards and tiny metadata.

03 · Safe area

Respect the device.

Bottom navigation, sheets and sticky actions include safe-area insets.

04 · Keyboard

Desktop is not mouse-only.

Focus order, shortcuts and visible focus follow the same hierarchy as the visual layout.

05 · Orientation

No landscape dependency.

Core mobile tasks complete in portrait; landscape enhances media but is never required.

06 · Content

Localisation survives.

Controls tolerate 30% text expansion and RTL without truncating meaning or reversing evidence chronology.

Stable navigation contract

The content moves. The rail does not.

Page position, active state and menu position are separate pieces of state. Selecting a section may update the URL and active marker, but must never reset or auto-scroll the left navigation.
